### `/content` 콘텐츠 & 학습

**4대 시스템** (브로슈어 p.2 그대로, 4-카드 그리드)

1. **감독형 온라인 모의고사**
   집에서 응시해도 긴장감 100%. 철저한 시험 감독 하에 진행되어 실제 고사장과 동일한 환경에서 실전 감각을 극대화합니다.

2. **양방향 라이브 해설**
   일방적인 인강은 그만. 강사진과 실시간으로 소통하며 헷갈렸던 문제와 개념을 그 자리에서 명쾌하게 해결합니다.

   단순한 점수 확인을 넘어, 매 시험마다 문항별 분석과 취약점 진단이 포함된 1:1 맞춤형 성적표를 제공합니다.

4. **수요일 밤 10시 30분 Q&A**
   심야의 밀착 케어. 공부하다 막히는 부분, 매주 수요일 밤 온라인 질문 클래스에서 즉시 해결합니다.

## 8. 작업 순서 제안

Claude Code에서 단계별로 진행:

1. **프로젝트 초기화**
   - `npx create-next-app@latest gr831-10days` (App Router, TypeScript, Tailwind)
   - GitHub repo 생성 + 첫 푸시
   - Vercel 프로젝트 연결

2. **디자인 시스템 구축**
   - Tailwind config에 컬러 팔레트 추가
   - 폰트 로딩 (Noto Serif KR, Pretendard)
   - 공통 컴포넌트 (Header, Footer, FloatingCTA)

3. **랜딩 페이지 (`/`) 우선 구현**
   - Hero + 핵심 숫자 + 미리보기 섹션
   - 이게 가장 임팩트 있어야 하므로 가장 먼저

4. **합격 실적 페이지 (`/admissions`) 구현**
   - 메인 임팩트 페이지
   - 카운트업 애니메이션 포함

5. **나머지 페이지 순차 구현**
   - `/about` → `/curriculum` → `/content` → `/honor` → `/faq` → `/yaksul` → `/universities` → `/instructors`

6. **모바일 반응형 점검**
   - 모든 페이지 모바일 폰트 큼직하게

7. **빌드 + 배포**
   - `npx next build` 빌드 테스트
   - 에러 없으면 git commit + vercel 배포
