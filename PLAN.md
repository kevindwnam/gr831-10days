## 3. 디자인 시스템

### 컬러 (순백 + 검정 + 빨강, 한정적 사용)

| 용도 | 컬러 | Hex |
|---|---|---|
| 페이지 배경 | 옅은 회색 (카드 대비) | `#F7F7F7` |
| 카드 배경 | 순백 | `#FFFFFF` |
| 본문 텍스트 | 검정 | `#000000` |

| 카드 보더/구분선 | 옅은 그레이 | `#E5E5E5` |

### 타이포그래피
- 헤더: Noto Serif KR (700, 900)
- 본문: Pretendard

### 모바일 우선
- 모바일 base font-size: 16~17px
- 컨테이너 최대폭 max-w-lg (모바일 카드 메뉴이므로 좁게)
- 데스크탑에서도 동일 컨테이너 폭 유지 (가운데 정렬, 양 옆 여백)

## 6. 공통 컴포넌트

### Header (단순)
- 가운데 또는 좌측에 십일전략 로고 (높이 36~40px)
- 로고 아래 작은 회색 텍스트: "GR831약술형논술연구소 온라인 브랜드"
- ⚠️ 네비게이션 메뉴 **없음** (카드가 곧 메뉴)
- sticky 또는 일반 흐름, 하단 옅은 보더만

### Footer (단순)
- 가운데 정렬, 작은 글씨
- "© 2026 GR831약술형논술연구소. All rights reserved."

### FloatingCTA
- 우측 하단 동그란 카카오 버튼 (z-50)
- 모든 페이지 표시

### MenuCard (신규 컴포넌트)
- 카드 형태: 흰 배경, 둥근 모서리(rounded-2xl), 옅은 그림자
- 구조: `[아이콘] [제목 / 부제] [>]`
- 호버/탭 시 살짝 위로 이동 + 그림자 강화
- 클릭하면 해당 라우트로 이동

## 7. 홈 페이지 상세 명세

### Hero 카드 구성
- 십일전략 로고 (상단)
- 메인 메시지: "약술형논술, 십일전략이 답입니다"
- 핵심 합격 숫자: "26학년도 가천대 135명 합격" (강조)
- 보조: "가천대 약술형논술 · 전국 합격률 1위"
- 추가 강조: "34명 보건간호계열 합격" (135명과 별개 라인)
- 강사진: "GR831 자문위원 출신 강사진 / 출제 의도를 가장 잘 아는 팀"
- 합격 역사 뱃지 (가로 배치): [24학년도 98명 합격] [25학년도 120명 합격]

스타일:
- 흰 카드, 상단 4px brand-red 라인
- 135명과 34명은 둘 다 강조 (font-serif, 큰 글씨, brand-red)
- 합격 역사 뱃지는 brand-dark 박스 + 흰 글씨
