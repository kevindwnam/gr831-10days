# 십일전략 (gr831-10days)

GR831약술형논술연구소의 공식 온라인 브랜드 **십일전략** 안내 페이지입니다.

## 기술 스택

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Noto Serif KR + Pretendard

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버: http://localhost:3000

## 빌드

```bash
npm run build
```
